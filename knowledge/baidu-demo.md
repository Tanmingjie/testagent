# 百度搜索 - 知识库

## 术语词汇

| 术语 | 说明 |
|------|------|
| 搜索框 | 百度首页搜索输入框 |
| 百度一下 | 搜索提交按钮 |
| 文库 | 百度文库入口 |
| 百科 | 百度百科入口 |
| 登录按钮 | 登录页面提交按钮 |

## 业务背景

百度是全球最大的中文搜索引擎，提供网页、图片、视频、学术等多种搜索服务。核心流程：用户打开首页 → 输入关键词 → 点击搜索 → 查看结果。

## 操作规范

1. 搜索前等待首页完全加载
2. 遇到百度安全验证标记为阻塞（BLOCK）

## 测试示例

```gherkin
场景: 百度首页搜索
  当 打开百度首页
  且 在搜索框中输入"test"
  且 点击"百度一下"按钮
  那么 页面显示搜索结果
```
