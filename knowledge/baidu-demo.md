## 业务背景

百度是全球最大的中文搜索引擎，提供网页、图片、视频、学术等多种搜索服务。核心流程：用户打开首页 → 输入关键词 → 点击搜索 → 查看结果。

登录场景：用户在百度首页点击"登录"进入 passport.baidu.com，完成身份验证后返回首页。

## 操作规范

1. **搜索前等待首页完全加载** — 首页加载后需等待热门搜索列表和搜索框渲染完成
2. **遇到百度安全验证标记为阻塞（BLOCK）** — 包括滑块验证码、文字点选等
3. **登录页处理流程**：
   - 百度登录页**默认展示二维码扫描界面**，不会直接显示用户名/密码输入框
   - 如果执行登录步骤且找不到账号密码输入框，必须先点击「用户名密码登录」切换标签
   - 用户名/密码输入框出现后，再执行输入操作
   - 如果步骤描述是"短信快捷登录"，点击「短信快捷登录」链接
4. **保持会话**：登录成功后跳转回首页，页面右上角会显示用户昵称

## 页面元素参考（Playwright Locator）

### 首页 (www.baidu.com)

| 操作 | 推荐定位器 |
|------|-----------|
| 点击"登录" | `page.getByRole('link', { name: '登录' })` |
| 在搜索框输入 | `page.getByRole('textbox')` — 首页只有一个 textbox |
| 点击"百度一下" | `page.getByRole('button', { name: '百度一下' })` |
| 点击顶部导航 | `page.getByRole('link', { name: '新闻' })` 等 |

### 登录页 (passport.baidu.com) — 初始状态（二维码模式）

| 操作 | 推荐定位器 |
|------|-----------|
| 切换到用户名密码登录 | `page.getByRole('paragraph', { name: '用户名登录' })` 或 `page.getByText('用户名登录')` |
| QQ 登录 | `page.getByRole('link', { name: 'QQ账号' })` |
| 微信登录 | `page.getByRole('link', { name: '微信' })` |
| 立即注册 | `page.getByText('立即注册')` |

### 登录页 — 用户名密码模式

| 操作 | 推荐定位器 |
|------|-----------|
| 输入用户名 | `page.getByPlaceholder('手机号/用户名/邮箱')` |
| 输入密码 | `page.getByPlaceholder('密码')` |
| 点击登录按钮 | `page.getByRole('button', { name: '登录' })` |
| 忘记密码 | `page.getByText('忘记密码？')` |
| 短信快捷登录 | `page.getByText('短信快捷登录')` |
| 切换回扫码 | `page.getByText('扫码登录')` |
